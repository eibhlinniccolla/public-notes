In [[JavaScript]], there are 7 [[primitive data types]]

- There are 7 primitive types
    - [[Undefined]]
    - [[String]]
    - [[Number]]
    - [[Boolean]]
    - [[Symbol]]
    - [[BigInt]]
    - [[Null]]
- [[Undeclared behaves a bit like a type, but is not listed as a type in the spec]]
- [[Variables don't have types, values do]]
