- Git
    - Check out a particular commit
        - `git log`
        - pick commit, copy hash
        - `git checkout <hash>`
    - Remove untracked files AND directories
        - `git clean -df`
    - Revert any uncommitted changes
        - `git reset HEAD --hard`
    - Return to latest commit
        - `git checkout <branchname>`
